# DFIR-Taller-Informatica-Forense
Repositorio oficial del Taller de Informática Forense (DFIR)  — Edición 2025. Material técnico del taller: laboratorios de análisis forense (memoria, disco y logs), procedimientos de adquisición y preservación, timeline analysis y reconstrucción de incidentes. Recursos diseñados para práctica directa y aplicación en entornos reales.
